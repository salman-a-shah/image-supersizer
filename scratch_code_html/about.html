About

ImageSupersizer is a free super-resolution service that doubles the resolution of an image while attempting to preserve image quality. This app achieves this through the use of a convolutional residual neural network.

This project is currently in alpha stage. Overtime, the supersizing model will be improved and additional models may be added for image enhancement.

If you find a bug, feel free to report it through the contact page.
